# H3 Hex Mapping Tester

A standalone browser app for testing Uber H3 hexagons on an interactive map.

## Features

- Render H3 cells around a center point
- Adjust H3 resolution
- Adjust `gridDisk` radius
- Recenter using latitude/longitude or map clicks
- Inspect cell ids and approximate cell area

## Run Locally

Use:

```powershell
.\start-h3-tester.cmd
```

Then open:

`http://localhost:8090/`
